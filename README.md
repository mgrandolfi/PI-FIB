# PI-FIB

Coursework repository for **PI (Protocolos de Internet)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository gathers the material developed for the course, including topic notes and a written assignment.

## Overview

PI is centered on Internet protocols and the technologies that make network communication possible.

This repository is mainly a structured archive of the subject content studied during the course, with topic summaries and a final written work related to the Dark Web, Deep Web, and Darknets.

## Repository structure

- `Topics 1 & 2.pdf` — course material for the first part of the subject
- `Topics 3, 4 & 5.pdf` — course material for the second part of the subject
- `Internet Protocols Written Work - Dark Web, Deep Web and Darknets.pdf` — written assignment
- `README.md` — repository overview

## What you'll find here

- Topic summaries and study material
- Protocol-related coursework
- Written technical work
- Structured reference material for review

## Focus

- Internet protocols
- Network communication
- Technical reading and writing
- Information technologies

## Notes

This is an academic repository tied to the PI coursework at FIB.

The goal of the repo is to keep the subject material organized, accessible, and easy to revisit.
