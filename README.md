# PI-FIB

Coursework repository for **PI (Protocolos de Internet)** at the **Facultat d'Informàtica de Barcelona (FIB - UPC)**.

This repository collects coursework, notes, and related material developed for the Internet Protocols subject.

## Overview

PI is part of the **Information Technologies** track and is centered on the concepts, technologies, and mechanisms that make modern Internet communication possible.

This repo acts as a structured collection of the material I developed during the course, including coursework deliverables, topic-based work, and supporting documentation.

## Contents

- Topic summaries and study material
- Written assignments (grade = 10)
- Protocol-related notes and exercises

## Focus

- Internet protocols
- Networking fundamentals
- Communication models
- Technical written work

## Notes

This is an academic repository tied to the PI coursework at FIB.

The main goal of the repo is to keep the subject material organized, accessible, and easy to review.
